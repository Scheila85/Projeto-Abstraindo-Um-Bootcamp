# Abstraindo um Bootcamp - Sistema em Java (POO)

## 📃Descrição

Projeto desenvolvido em Java com o objetivo de aplicar os principais conceitos de Programação Orientada a Objetos (POO). 

Para a construção deste projeto, foram criadas e modeladas as classes Bootcamp, Cursos, Mentorias, Developers e Administrador, utilizando atributos e métodos que representam o domínio do sistema.

Foram aplicados conceitos que representam os pilares da POO - Abstração, Encapsulamento, Herança e Polimorfismo - com o objetivo de melhorar a organização da estrutura do projeto e facilitar a manutenção do código.

Além disso, foram utilizados recursos como **Collections** e **Stream API** para o armazenamento e manipulação de conjuntos de dados.

---

## 🎯 Principais funcionalidades:

- Gerenciamento de cursos e mentorias;
- Inscrição e controle dos conteúdos inscritos e concluídos;
- Manipulação e consulta de dados utilizando Collections e Stream API.
